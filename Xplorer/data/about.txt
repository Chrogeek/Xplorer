Third Party Tools and Libraries

1. Direct2D: A hardware-based graphics engine developed by Microsoft, compatible with Microsoft Windows 7 (and later). Much more complex than GDI, but also much more efficient.

2. nlohmann/json (https://github.com/nlohmann/json): for parsing JSON data. MIT License.

3. BASS.dll (https://www.un4seen.com/bass.html): an audio library developed by Un4seen Developments. This software utilize this library with a free, non-commercial license.

4. bjorn/tiled (https://github.com/bjorn/tiled): an open-source visual map editor, supports XML and json as its file format.
